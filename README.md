# Gestor de Facturas

App para gestionar facturas de compras y ventas con cálculo automático de IVA 21%.

## Instalar como App (Más fácil)

1. Sube esta carpeta a GitHub
2. Activa GitHub Pages en Settings > Pages
3. Abre la URL de GitHub Pages
4. Busca el icono de instalación en la barra de direcciones
5. Clic en el icono y sigue las instrucciones

## La app incluye:

- Gestión de facturas de compra y venta
- Cálculo automático de IVA 21%
- Subir PDFs de facturas
- Generar reportes en PDF
- Funciona sin internet
- Datos guardados localmente

## Subir a GitHub (solo 3 pasos)

1. Crea un repositorio en GitHub.com
2. Ejecuta `subir_github.bat`
3. Pega la URL del repositorio cuando se te pida

## Activar GitHub Pages

1. Ve a tu repositorio en GitHub
2. Clic en Settings > Pages
3. En "Source" selecciona "main" y guarda
4. Espera 1-2 minutos
5. Tu app estará en: https://TU_USUARIO.github.io/NOMBRE/
