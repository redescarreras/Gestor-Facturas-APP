# Historial de Cambios

## [1.0.0] - 2026-01-07

### Añadido
- Dashboard principal con resumen de finanzas
- Gestión de facturas de compra y venta
- Cálculo automático de IVA 21%
- Subida y visualización de PDFs de facturas
- Generación de reportes mensuales, trimestrales y anuales
- Exportación de reportes a PDF
- Almacenamiento local con IndexedDB
- Sistema de backup y restauración
- Diseño responsive para móviles
- Colores corporativos personalizados

### Características
- Interfaz moderna y profesional
- Navegación lateral con secciones
- Cálculo en tiempo real del IVA
- Persistencia de datos en navegador
- Sin necesidad de servidor externo

### Instalación
- Script de instalación para Windows (install.bat)
- Script de desinstalación para Windows (uninstall.bat)
- Servidor local Python opcional
- Scripts para publicación en GitHub

### Primera versión
- Estructura básica de la aplicación
